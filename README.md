# Phishing-URL-Detection-Using-Deep-Learning
Deep learning-based phishing URL detection system using attention-based feature fusion to identify malicious and previously unseen URLs with improved accuracy.

# Implementation
The project is implemented using a Jupyter Notebook (Google Colab) and follows a structured pipeline:
- Data loading and initial exploration of the URL dataset  
- Data preprocessing including handling missing values and encoding labels  
- Graph construction based on URL features to capture relationships between data points  
- Feature extraction and transformation for model input  
- Model training using machine learning techniques  
- Evaluation of model performance using accuracy and other metrics  

# Dataset
The dataset contains labeled URLs categorized as phishing or legitimate. It is used for training and testing the model.A reduced sample of the dataset is included for demonstration purposes.

# Technologies Used
- Python
- Google Colab (Jupyter Notebook)
- Pandas, NumPy
- Scikit-learn / TensorFlow

# How to Run
1. Open the notebook in Google Colab or Jupyter Notebook
2. Upload the dataset if required
3. Run all cells step by step to reproduce results

# Results
The model achieves good accuracy in detecting phishing URLs and performs better than traditional methods for unseen data.
